## App Management using VMC

### List, Push, Update and Delete Applications

    vmc apps
    vmc push
    vmc update <app_name>
    vmc delete <app_name>

### Start/Restart/Stop Applications

    vmc start <app_name>
    vmc restart <app_name>
    vmc stop <app_name>

### View logs and crashlogs for applications

    vmc logs <app_name>
